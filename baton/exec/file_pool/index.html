Input file pool collection from CLI sources.

Gathers InputFiles from positional arguments, --diff <refspec> (via git diff --name-only), and --files <path|-> (newline-delimited list). Directories are walked recursively. The resulting pool is deduplicated by canonical (absolute, symlink-resolved) path.
